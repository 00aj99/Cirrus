Buttons

Cirrus provides minimally designed buttons created to work well with any website. It is also minimally designed to provide a blank canvas for designers to craft custom buttons for any design.

Basic options

Non animated buttons.

test
<div class="btn">test</div>
<button>Regular Button</button>
<input type="submit" value="Submit"/>

Animated

test
<div class="btn animated">test</div>
<button class="animated">Regular Button</button>
<input type="submit" class="animated" value="Submit"/>
States

Disabled buttons.

Please note that using <div class="btn">test</div> is not supported. 

<button disabled class="animated">Regular Button</button>
<input disabled type="submit" class="animated" />
